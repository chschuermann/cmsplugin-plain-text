Documentation
-------------

The full documentation is at https://cmsplugin-plain-text.readthedocs.org.

Quickstart
----------

Install cmsplugin-plain-text::

    pip install cmsplugin-plain-text

Then use it in a project::

    import cmsplugin_plain_text
